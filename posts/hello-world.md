## 关于这个博客

这个博客支持直接放 Markdown 文件！只需要把 .md 文件放到 `posts/` 目录，**无需手动更新索引**，GitHub Actions 会自动扫描并生成文章列表。

## 特性

- ✨ 支持标准 Markdown 语法
- 📝 自动渲染为 HTML
- 🏷️ 自动提取日期和标签
- 🖼️ 支持图片
- 🤖 **全自动索引** - 只需 push 就完事

**加粗文字** 和 *斜体文字* 都支持～
